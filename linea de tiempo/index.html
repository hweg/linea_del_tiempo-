Línea del tiempo: Navegadores web

Breve resumen
1990-WWW World Wide Web: El primer navegador fue creado en 1990 por Tim Berners-Lee, el padre de la WWW (World Wide Web) y se llamó igual, **WorldWideWeb**, si bien luego se rebautizó como Nexus.
1993-ViolaWWW ViolaWWW es considerado como el primer navegador gráfico de la historia, precursor del popular navegador Mosaic
1993-Mosaic Mosaic fue el segundo navegador gráfico disponible para visualizar páginas web y el primer navegador gráfico para Microsoft Windows
1994-Spyglass Spyglass fue una compañía (NASDAQ SPYG) de software de internet con central en Champaign, Illinois. Fundada en 1990
1994-NetScape Navigator Netscape Navigator fue un navegador web y el primer producto comercial de la compañía Netscape Communications, creada por Marc Andreessen.
1995-Internet Explorer Internet Explorer mejor conocido como IE, es un navegador web desarrollado por Microsoft para el sistema operativo Microsoft Windows desde 1995.
1996-Opera El navegador comenzó en 1994 como un proyecto de investigación en Telenor, una empresa de telecomunicaciones noruega.
2003-Apple Safari La primera versión beta de Safari fue presentada en la exposición Macworld el 7 de enero de 2003 y fue liberada en forma de beta pública.
2003-Mozilla Firefox Mozilla Foundation es una organización sin ánimo de lucro dedicada a la creación de software libre. Tiene como misión "mantener la elección y la innovacion en internet
2008-Google Chrome Google Chrome es un navegador web desarrollado por Google y compilado con base en varios componentes e infraestructuras de desarrollo de aplicaciones de codigo abierto y disponible gratuitamente
2012-Yandex Yandex Browser es un navegador web gratuito desarrollado por la corporación rusa de búsqueda web Yandex que utiliza el motor de renderizado Blink utilizado en el proyecto de código abierto Chromium.
2015-Vivaldi Vivaldi es un navegador web desarrollado por Vivaldi Technologies, una compañía fundada por el cofundador y anterior ceo de opera, Jon Stephenson von Tetzchner.​ El eslogan del proyecto es Un nuevo navegador para nosotros y nuestros amigos.​
